## Quick Start

1. Clone this repo:

    `git clone https://github.com/DanAtlas/boilerplate.git`

2. Install all packages:

    `npm i`

3. Run project in a development mode:

    `npm start`

4. Open your browser with the next address:

    `http://localhost:3000/`

## Available Scripts
  1. `npm start`

  2. `npm run build`

  3. `npm lint`
